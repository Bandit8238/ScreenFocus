⚙️ How It Works

FocusOverlay applies a visual and functional layer over your workspace:

Adds a blur effect to active content
Displays a centered focus overlay
Pauses all media elements automatically
Mutes audio playback for distraction control

It’s designed to be lightweight and run entirely in the browser with no dependencies.
